# Reusable AI prompts

Copy-paste prompts for common workflow tasks. Each file contains a fenced block ready for a new chat.

| Prompt | Use when |
|--------|----------|
| [new-project-setup.md](./new-project-setup.md) | First session after `workflow init` (empty / greenfield) |
| [project-alignment.md](./project-alignment.md) | **Align template to existing docs and/or code** |
| [orchestrator.md](./orchestrator.md) | Multi-step or multi-role work |
| [feature-development.md](./feature-development.md) | Single feature with TDD |
| [ingest-second-brain.md](./ingest-second-brain.md) | Processing `second-brain/raw/` |

All prompts assume **`AGENT.md`** is the canonical instruction file.
